2.如何做横向布局

<li style="float:left">

注释：会有bug 因为它本身自带样式

用本行代码消除样式

<ul style="list-style: none; margin= 0" class="clearfix">

用float时会有bug

在css文件做个clearfix类,将下列代码写到所有float元素的爸爸上面

.clearfix::after{
    content:'';
    display: block;
    clear: both;
}
